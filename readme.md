

## Server Requirements

 * git
 * go
 * EC2 Role with access to:
   * KMS:Decrypt (credentials)
   * KMS:Encrypt (data)
   * KMS:Decrypt (data)
   * S3:PutObject      
   * S3:ListBucket      
   * S3:GetObject      

### GO bin config

```bash
# .bash_profile/.profile
# ...
export GOPATH=$HOME/go
export PATH=$PATH:/usr/local/go/bin:$GOPATH/bin
```

## Install/Update

```bash
go get -u github.com/SmallBusinessVictoria-ABR-ETL/tools
```
